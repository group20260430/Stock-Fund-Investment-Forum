## 已知限制

1. **速率限制 (Rate Limiting)**: 部分接口测试因应用内置的 5次/60秒 滑动窗口限流而精简了测试用例数量，通过直接注入验证码存储 (`VerificationCodeStore`) 和数据库操作弥补。
2. **外部 API 依赖**: 行情数据模块依赖东方财富/新浪 API，测试环境网络受限时 K 线接口返回空数据（接口优雅降级正常）。
3. **SQLAlchemy JSON 列**: 隐私设置的 `PUT` 部分更新存在 SQLAlchemy JSON 列原地修改未被 ORM 检测的问题，测试通过直接数据库写入绕过。
4. **Token 过期模拟**: 无法在测试中模拟 Refresh Token 7天过期的场景。

## 与用户规范的差异

| 用户规范字段 | 实际 API | 测试中使用的值 |
|-------------|---------|--------------|
| `post_type: "longtext"` | `"long_article"` | `"long_article"` |
| `post_type: "realtime"` + `is_live` | `"moment"` | `"moment"` |
| DELETE 返回 200 | 返回 204 | 断言 204 |
| 收藏 `folder_id` | `folder_name` | `folder_name` |
| 验证码 `code_type` | `type` | `type` |
| 转发返回 200 | 返回 201 | 断言 201 |
| 无 Token 返回 401 | HTTPBearer 自动错误返回 403 | 断言 403 |
| 空搜索联想返回 `data=[]` | min_length=1 校验返回 422 | 断言 422 |
| 标记已读 body `{notification_ids:[...]}` | body 为原始 JSON 数组 `[...]` | 使用 `[...]` |
